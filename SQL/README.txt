1. Install MAMP
https://www.mamp.info/en/downloads/

2. Run MAMP installer and launch MAMP

3. After starting MAMP, verify that both the Apache Server and the MySQL Servers have started (indicated by green light) 

4. Click on Open WebStart Page

5. Click on phpMyAdmin

6. At the bottom of the screen, there is a "console".

7. Copy the contents of this file into the console and press ctrl+enter:
https://github.com/johnsliao/stock-kings/blob/master/SQL/1_StockKings_CreateDB.sql

The StockKings DB has been built!
